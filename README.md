## 🇬🇧 English

Welcome to the **Jtool-io** repository by Pibiger Technology. This repository contains the software, drivers, and documentation for the Pibiger USB IO Series products, specifically the **JIO Module** and **JI2C Module**.

### 🌟 Overview

The Pibiger USB IO Series provides powerful and easy-to-use USB-to-GPIO, PWM, ADC, and I2C solutions. These modules are designed to help developers, makers, and engineers quickly interface with hardware components directly from a host computer via USB.

### 📦 Repository Contents

This repository is organized into the following main sections:

#### 1. JIO_Module
The JIO Module is a versatile USB tool for controlling GPIO, PWM, and reading ADC signals.
- **JIO.EXE**: Excellent Windows software for independent configuration of multiple channels. It allows for direct driving of GPIO, PWM, ADC, and differential ADC signals from a host computer.
- **JIO User Manual V10.pdf**: Comprehensive hardware and software user manual.
- **JIO1.2.0.zip**: The latest software release package.

#### 2. JI2C_Module
The JI2C Module is designed specifically for I2C communication and EEPROM programming.
- **JI2C.EXE**: Windows software tailored for I2C device interaction.
- **JI2C Module User Manual**: Detailed instructions on how to use the JI2C hardware and software.
- **JI2C.zip**: The software release package.

#### 3. JTool (Development SDK)
For advanced users who want to integrate JIO and JI2C functionalities into their own applications.
- **Command Line Interface (CMD)**: Tools for automating tasks using scripts.
- **DLL Libraries**: Dynamic Link Libraries provided for secondary development in **C#** and **Python**.
- **JTool_UserManual_EN_V10.pdf**: English user manual for the development SDK.
- **JTool.zip**: The complete SDK package.

### 🚀 Getting Started

1. **Download the Software**: Navigate to the respective module folder (`JIO_Module` or `JI2C_Module`) and download the `.zip` file.
2. **Read the Manual**: Open the provided PDF user manuals to understand the hardware connections and software interface.
3. **Run the Application**: Extract the zip file and run the `.EXE` application on your Windows machine.
4. **Develop Your Own App**: If you wish to write your own code, check out the `JTool` folder for DLLs and Python/C# examples.

## 🇨🇳 中文

欢迎访问 Pibiger Technology 的 **Jtool-io** 仓库。本仓库包含了 Pibiger USB IO 系列产品（特别是 **JIO 模块** 和 **JI2C 模块**）的软件、驱动程序和相关文档。

### 🌟 简介

Pibiger USB IO 系列提供了强大且易于使用的 USB 转 GPIO、PWM、ADC 和 I2C 解决方案。这些模块旨在帮助开发者、创客和工程师通过 USB 接口直接从上位机快速连接和控制硬件组件。

### 📦 仓库内容

本仓库主要包含以下几个部分：

#### 1. JIO_Module (JIO 模块)
JIO 模块是一款多功能的 USB 工具，用于控制 GPIO、PWM 以及读取 ADC 信号。
- **JIO.EXE**: 优秀的 Windows 客户端软件，支持多通道独立配置。允许上位机直接驱动 GPIO、PWM、ADC 以及差分 ADC 信号。
- **JIO User Manual V10.pdf**: 详尽的硬件和软件用户手册。
- **JIO1.2.0.zip**: 最新版本的软件发布包。

#### 2. JI2C_Module (JI2C 模块)
JI2C 模块专为 I2C 通信和 EEPROM 编程设计。
- **JI2C.EXE**: 专为 I2C 设备交互定制的 Windows 软件。
- **JI2C Module User Manual**: 详细的 JI2C 硬件和软件使用说明。
- **JI2C.zip**: 软件发布包。

#### 3. JTool (开发 SDK)
专为希望将 JIO 和 JI2C 功能集成到自有应用程序中的高级用户提供。
- **命令行工具 (CMD)**: 用于通过脚本实现任务自动化的工具。
- **DLL 动态链接库**: 提供用于 **C#** 和 **Python** 二次开发的库文件。
- **JTool_UserManual_EN_V10.pdf**: 开发 SDK 的英文用户手册。
- **JTool.zip**: 完整的 SDK 开发包。

### 🚀 快速入门

1. **下载软件**: 进入对应的模块文件夹（`JIO_Module` 或 `JI2C_Module`），下载 `.zip` 压缩包。
2. **阅读手册**: 打开提供的 PDF 用户手册，了解硬件连接方式和软件界面操作。
3. **运行程序**: 解压下载的文件，在 Windows 系统上运行 `.EXE` 应用程序。
4. **自主开发**: 如果您希望编写自己的控制代码，请查看 `JTool` 文件夹获取 DLL 文件以及 Python/C# 的开发示例。
